# Forge

Forge is a private, friends-only competitive execution arena.

This first build is a mobile-first React prototype with:

- Arena leaderboard
- Weekly live pot display
- Stakes room and raise slider
- Sunday submission terminal
- Monday reveal loop
- AI judge prompt seed

## Run locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:5173
```

## Build

```bash
npm run build
```
